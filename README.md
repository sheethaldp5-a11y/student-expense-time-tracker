# Smart Student Expense & Time Leakage Tracker

A Python-based productivity and expense tracking system designed for students to monitor daily spending, track time usage, and analyze productivity leaks.

## 🚀 Features
- Track daily expenses
- Monitor time usage
- Store student data using SQLite database
- Generate productivity insights
- Simple and lightweight Python implementation

## 🛠️ Tech Stack
- Python
- SQLite
- File handling & data analysis

## 📂 Project Structure
app.py – main program  
database.py – database operations  
expense.py – expense tracking  
time_log.py – time monitoring  
analysis.py – productivity analysis  

## ▶️ How to Run
```bash
python app.py
